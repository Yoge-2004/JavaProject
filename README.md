## ğŸ› ï¸ Getting Started

To run or edit the code locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Yoge-2004/LMSJavaFX.git
   cd LMSJavaFX
   ```

2. **Open in your favorite IDE**

   - Using Maven:  
